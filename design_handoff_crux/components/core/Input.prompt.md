Labelled text field or textarea. Focus shows the violet ring.

```jsx
<Input label="Case title" placeholder="Why is my pace dropping?" />
<Input label="Paste the messy problem" multiline placeholder="Dump everything you know…" />
<Input label="Target metric" mono placeholder="VO2 < 48" />
```

- `multiline`: textarea (use for the New Case paste box)
- `mono`: switch to the mono data register for numeric / ID entry
- `label`, `hint`: optional surrounding text
