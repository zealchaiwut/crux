A cited source on a Plan's evidence — one chip per real citation. The colored icon marks the kind.

```jsx
<SourceChip kind="article" href="https://...">Daniels, Running Formula</SourceChip>
<SourceChip kind="book">Noakes — Lore of Running</SourceChip>
<SourceChip kind="youtube">CritFit · Deload science</SourceChip>
```

- `kind`: `book` (amber) · `article` (blue) · `youtube` (red)
- Pass `href` to make it an external link. Never fabricate citations — one chip = one real source.
