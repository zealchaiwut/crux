The cheapest decisive test for the leading plan: a type chip, one big mono target metric, and a cost/time foot line.

```jsx
<ProbeCard
  type="lab-test"
  targetMetric="ferritin > 30 ng/mL"
  status="designed"
  cost="$40"
  time="3 days"
  note="Order a ferritin + iron panel — see a doctor, crux does not run this."
/>

<ProbeCard type="prototype" targetMetric="blurt-rate ↓ 50%" cost="1 sprint" time="2 wks"
  onSendToCommander={() => copySpec()} />
```

- `type`: `measurement` · `lab-test` · `behaviour-experiment` · `prototype`
- Only `prototype` shows "Send to commander". Be honest — name the real action for non-app probes.
- `targetMetric` is the single number that settles the case.
